# Scalabl · Demos

Demos funcionales anonimizadas de IA aplicada a la operación, para uso en clase/presentaciones.
Todas con datos ficticios — ninguna empresa, cuenta o dato es real. Sin backend (client-side).

- `/` — hub
- `/pagos` — Excel de pagos → lote TXT Interbanking
- `/facturas` — factura vs. orden de compra (detección de diferencias)
- `/trazabilidad` — trazabilidad de lotes (historial verificable)

Stack: HTML/JS estático + SheetJS (`/lib/xlsx.full.min.js`). Deploy: Vercel.
